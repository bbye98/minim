LyricsAPISession#

class minim.spotify.LyricsAPISession(*, sp_dc: Optional[str] = None, access_token: Optional[str] = None, expiry: Optional[datetime] = None)[source]#

Bases: _Session

A Spotify Lyrics API session.

Parameters:
sp_dcstr, keyword-only, optional

Spotify Web Player sp_dc cookie.

access_tokenstr, keyword-only, optional

Access token. If provided, the token exchange is bypassed.

expirydatetime.datetime, keyword-only, optional

Expiry time of access_token. If provided, the user will be reauthenticated using the default authorization flow (if possible) when access_token expires.

Attributes:
API_URLstr

URL for the Spotify Lyrics API.

TOKEN_URLstr

URL for access token requests.

sessionrequests.Session

A session object with persisting headers.
