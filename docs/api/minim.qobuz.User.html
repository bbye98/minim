Get audio stream data for all tracks in an album or a playlist.

User authentication

Requires user authentication and an active Qobuz streaming plan for full, high-quality audio streams.

Note

This method is provided for convenience and is not a Qobuz API endpoint.

Parameters:
collectionint, str, minim.qobuz.Album, or minim.qobuz.Playlist

Qobuz collection ID or object.

typestr, optional

Qobuz collection type. Required if collection is not a Qobuz album or playlist object.

Valid values: "album" and "playlist".

qualitystr, keyword-only, default: 27

Maximum audio quality.

Valid values:

  • 5 or "MP3" for constant bitrate (320 kbps) MP3.

  • 6 or "CD" for CD-quality (16-bit, 44.1 kHz) FLAC.

  • 7 for up to 24-bit, 96 kHz Hi-Res FLAC.

  • 27 or "HI-RES" for up to 24-bit, 192 kHz Hi-Res FLAC.

savebool, keyword-only, default: False

Determines whether the streams are saved to audio files.

pathstr, keyword-only, optional

If save=True, path in which the audio files are saved.

folderbool, keyword-only, default: False

Determines whether a folder in path (or in the current directory if path is not specified) is created to hold the audio files.

metadatabool, keyword-only, default: True

Determines whether the audio filesâ€™ metadata is populated.

Returns:
streamslist

Audio stream data. If save=True, the streams are saved to audio files and their filenames are returned instead. If collection is an Album or Playlist object, the stream data is stored in its underlying Track objectsâ€™ stream instance variables instead.

get_track_credits(track: Optional[Union[int, str, Track]] = None, *, performers: Optional[str] = None, roles: Optional[list] = None) → Union[None, dict[str, list]][source]#

Get credits for a track.

Note

This method is provided for convenience and is not a Qobuz API endpoint.

Parameters:
trackint, str, or minim.qobuz.Track

Qobuz track ID or object.

performersstr, keyword-only, optional

An unformatted string containing the track credits obtained from calling Session.get_track().

roleslist, keyword-only, optional

Role filter. The special "Composers" filter will combine the "Composer", "ComposerLyricist", "Lyricist", and "Writer" roles.

Note

roles is case-sensitive; all values must be in Pascal case.

Valid values: "MainArtist", "FeaturedArtist", "Producer", "Co-Producer", "Mixer", "Composers" ("Composer", "ComposerLyricist", "Lyricist", "Writer"), "MusicPublisher", etc.

Returns:
creditsdict

A dictionary containing the track contributors, with their roles (in snake case) being the keys. If track is a Track object, the information is stored in its credits instance variable instead.
