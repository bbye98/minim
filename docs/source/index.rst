Minim
=====

Minim is a lightweight Python 3 library for interacting with popular 
music service APIs—Deezer, Discogs, iTunes, Musixmatch, Qobuz, Spotify, 
and TIDAL—and for manipulating audio files, such as updating metadata 
and converting between audio formats.
